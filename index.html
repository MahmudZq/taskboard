1. Selecting by ID

Using getElementById

Returns a single element that matches the given id.


    
javascript

    


const element = document.getElementById('myElement');
console.log(element);


const element = document.getElementById('myElement');
console.log(element);

Creating an Element

We can create an element using the document.createElement() method.


    
javascript

    


const newDiv = document.createElement('div');  // element created
newDiv.innerText = "Hello, I'm a new div!"; // content added                                        
newDiv.classList.add('new-class'); // class added to the element  
newDiv.style.color = 'blue';  // style added to the element  


const newDiv = document.createElement('div'); // element created
newDiv.innerText = "Hello, I'm a new div!";   // content added
newDiv.classList.add('new-class');            // class added to the element
newDiv.style.color = 'blue';                  // style added to the element

Add an Element

Using appendChild()

Adds the element as the last child of a parent element.


    
javascript

    


document.body.appendChild(newDiv); // Add newDiv to the body


document.body.appendChild(newDiv); // Add newDiv to the body

Using append()

Allows adding multiple elements or text at once.


    
javascript

    


document.body.append("Text before div", newDiv); // Add Text & newDiv to the body at once.


document.body.append("Text before div", newDiv); // Add Text & newDiv to the body at once.

Remove an Element

We can remove an element using remove() or removeChild().

Using remove(), the Modern method. 


    
javascript

    


newDiv.remove(); // Removes the element directly


newDiv.remove(); // Removes the element directly

Using removeChild(), the Older method. 


    
javascript

    


const parent = document.getElementById('parentDiv');
const child = document.getElementById('childDiv');
parent.removeChild(child);  // Removes childDiv from parentDiv


const parent = document.getElementById('parentDiv');
const child = document.getElementById('childDiv');
parent.removeChild(child); // Removes childDiv from parentDiv
